# Visao Geral do Sistema
**Atualizado em:** 20/04/2026

## Estagio atual
- Backend Laravel com dominio inicial de RH e eSocial ja modelado.
- Autenticacao, dashboard e perfil prontos como base administrativa.
- Modulo de servidores com listagem operacional, admissao inicial, detalhe e edicao do cadastro.
- Modulo de lotacoes com listagem, cadastro e edicao da estrutura organizacional.
- Modulos de cargos e funcoes com operacao inicial para sustentar a base estrutural do RH.
- Rastreabilidade de eventos eSocial ja existe no modelo `EventoEsocial`, mas ainda sem transmissao real.

## Proposta de produto
- Plataforma web para operacao administrativa de RH em orgaos publicos.
- Interface para cadastros, admissoes, atualizacoes e acompanhamento operacional.
- Backend preparado para validar dados, gerar payloads e evoluir para integracao com o governo.

## Direcao funcional
- Primeiro consolidar cadastros estruturais e fluxo de admissao.
- Depois ampliar operacoes de RH sobre o vinculo funcional.
- Em paralelo, amadurecer o modulo de eventos eSocial e a rastreabilidade.
