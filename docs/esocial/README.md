# eSocial
**Atualizado em:** 30/04/2026

## Objetivo
Organizar a evolucao funcional do FolhaNova como plataforma web de RH para orgaos publicos, mantendo aderencia ao eSocial e separando prioridade funcional, modelagem e backlog tecnico.

## Referencia oficial usada nesta rodada
- Portal oficial de Documentacao Tecnica do eSocial: https://www.gov.br/esocial/pt-br/documentacao-tecnica/documentacao-tecnica
- Em 30/04/2026, o portal lista a versao S-1.3 com referencia a `NT 06/2026 rev. 09/04/2026`, esquemas XSD ate `NT 06/2026` e MOS `consolidado ate NO 10/2026`.
- A explicacao humana da futura integracao por Web Services esta em `integracao-api-esocial.md`.
